# MoveData python utility

Утилита для windows выполняет следующее: 
1. копирует папку из каталога BASE в другой каталог BASENEW, который находится также на этом же диске;
2. после копирования папки проверяет целостность папки в каталоге BASENEW;
если папка в BASENEW скопирована без ошибок, то скрипт удаляет ее из каталога BASE и оставляет эту папку в каталоге BASENEW;
3. и этот порядок должен быть применен ко всем папкам, которые находятся в каталоге BASE;
4. также в каталоге BASE есть один текстовый файл небольшого размера, который не находится в других папках. Этот файл также должен быть скопирован в каталог BASENEW и удален из каталога BASE.
Дополнительные требования:
- возможность логирования и последующего чтения логов;
- возможность остановки скрипта, но после получения команды об остановке скрипт должен завершить копирование текущей папки, проверить ее целостность и удалить из каталога BASE и лишь затем остановится.

## Инициализация проекта и установка зависимостей

Старт проекта: `uv init --python=3.12.10`.
Настройка виртуального окружения: `uv sync`.

Для разработки применялись: 
- python 3.12.10;
- pyside6;
- uv;
- ruff;
- pytest;
- pre-commit;
- loguru.

Для установки зависимостей использовались команды:
```powershell
uv add PySide6 loguru
uv add --dev pytest pytest-qt ruff pre-commit
```

## Использование pre-commit
1. Добавить файл с конфигурацией .pre-commit-config.yaml.
2. Запустить команду uv run pre-commit install
3. Запустить команду uv run pre-commit run --all-files
